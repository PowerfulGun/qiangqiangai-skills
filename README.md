# qiangqiangai-skills
枪枪AI的 skills 库（偏 Windows 系统工具类）

## 技能列表

| 技能 | 说明 |
|------|------|
| [系统体检](skills/system-check/SKILL.md) | 电脑体检、全面检查：硬件信息、CPU/内存占用、磁盘健康、启动项、系统更新，输出结构化报告 |
| [Windows 服务管理](skills/windows-service/SKILL.md) | 查看/启动/停止/禁用 Windows 服务，设置启动类型，安全提示 |
| [Windows 网络与防火墙](skills/windows-network-firewall/SKILL.md) | 网络状态、网卡与 IP、端口监听、防火墙规则、DNS 与连通性检查 |
| [Windows 计划任务](skills/windows-task-scheduler/SKILL.md) | 查看/创建/禁用计划任务，触发器与最近运行结果，创建任务示例命令 |
| [磁盘与临时文件清理](skills/disk-cleanup/SKILL.md) | 磁盘空间总览、可清理项（Temp/回收站）、大文件/大目录扫描建议 |
| [注册表与右键菜单](skills/registry-context-menu/SKILL.md) | 安全读取注册表、添加「用 Cursor 打开」等右键菜单（仅提供 .reg/脚本，不直接改注册表） |

> 说明：GitHub 上暂无热门的、现成可用的「Windows 系统工具类」SKILL 仓库，以上技能按本库格式自建，便于在 Windows 上做系统排查与常用操作。
